## 开发计划

### 技术栈确认
- **后端**: FastAPI + yt-dlp (Python API 直接调用，不改动开源源码，纯封装)
- **前端**: Vue3 + Vite + Pinia (状态管理) + 原生 CSS (双主题变量)
- **实时进度**: WebSocket / SSE 推送下载进度
- **存储**: 服务器临时目录下载 → 提供下载链接 (适配手机/桌面)

### 架构设计

```
┌─────────────────────────────────────────────────────────┐
│  前端 Vue3 (Vite)                                         │
│  ┌──────────────┐    ┌──────────────────────────────┐    │
│  │ 左侧输入控制区 │ ←→ │ 右侧实时预览结果区             │    │
│  │ - 多行输入框   │    │ - 封面+信息卡片                │    │
│  │ - 拖拽导入     │    │ - 可视化画质选择器             │    │
│  │ - 迷你标签列表 │    │ - 下载/音频双按钮              │    │
│  └──────────────┘    └──────────────────────────────┘    │
│  侧滑历史抽屉 | 双主题切换 | 移动端底部悬浮栏              │
└───────────────────────────┬─────────────────────────────┘
                            │ HTTP / WebSocket
┌───────────────────────────┴─────────────────────────────┐
│  后端 FastAPI                                             │
│  - POST /api/info     解析视频信息(标题/封面/画质列表)    │
│  - POST /api/download 创建下载任务 → task_id              │
│  - WS   /ws/{task_id} 推送下载进度                        │
│  - GET  /api/file/{id} 下载文件                           │
│  - GET  /api/history  历史记录                            │
│  ┌────────────────────────────────────────────────┐      │
│  │ yt-dlp 封装层 (extract_info / download)          │      │
│  └────────────────────────────────────────────────┘      │
└─────────────────────────────────────────────────────────┘
```

## 📌 使用说明

### 本地运行
```bash
# 方式1：使用启动脚本（推荐）
./start.sh

# 方式2：手动启动
# 终端1 - 启动后端
cd backend
source venv/bin/activate
uvicorn main:app --host 0.0.0.0 --port 8000 --reload

# 终端2 - 启动前端
cd frontend
npm run dev
```

### Docker部署
```bash
# 一键启动
docker-compose up -d

# 查看日志
docker-compose logs -f

# 停止服务
docker-compose down
```

### 访问地址
- **前端界面**: http://localhost:3000 (开发) / http://localhost (Docker)
- **后端API**: http://localhost:8000
- **API文档**: http://localhost:8000/docs
