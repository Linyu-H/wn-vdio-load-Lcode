## 🚀 使用指南

### 快速启动（3种方式）

#### 方式1: 一键启动脚本（推荐）
```bash
./start.sh
```

#### 方式2: 手动启动
```bash
# 后端
cd backend && source venv/bin/activate
uvicorn main:app --host 0.0.0.0 --port 8000 --reload

# 前端（新终端）
cd frontend && npm run dev
```

#### 方式3: Docker（生产环境）
```bash
docker-compose up -d
```

### 访问地址
- **前端**: http://localhost:3000 (开发) / http://localhost (生产)
- **后端**: http://localhost:8000
- **API文档**: http://localhost:8000/docs
